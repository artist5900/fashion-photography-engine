# Changelog

## v0.2.0

Fashion Photography Engine v0.2.0 turns the project from a starter prompt database into a simple end-to-end planning workflow.

### Added

- Prompt Builder for production-ready fashion photography prompts
- Photography Planner for ranked shot plans
- Planner compatibility rules and validation
- Plan to Prompts exporter for converting lookbook plans into Markdown prompts
- Garment Parser v1 schema, examples, and rule-based parser
- Shot, pose, scene, brand style, and lens databases
- Example lookbook request, plan, and generated prompt set

### Changed

- README now reflects the current project milestones and release status
- Version updated to 0.2.0
